# Realtime Store Data Engine

## Features
- Multi-category scraping
- Multi-page scraping
- MySQL storage
- CLI interface
- Search, filter, stats, insert, filter by price, filter by rating, show categories, sort by price

## Tech Stack
- Python
- BeautifulSoup
- MySQL
- Requests

## How to Run
1. Install requirements
2. Configure database
3. Run cli.py

This project is for learning purpose
