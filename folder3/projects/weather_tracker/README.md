# Weather Tracker API Project

## Overview
A Python project that fetches live weather data from the OpenWeatherMap API, stores it in MySQL and provides analytics.

## Features
- Fetch live weather for any city
- Store historical weather data in MySQL
- CLI interface for real-time interaction
- Analytics using Pandas

## Tech Stack
- Python
- Requests
- MySQL
- Pandas
- dotenv (for API keys & credentials)

## How to Run

1. Install dependencies:
2. Requests, mysql-connector-python, pandas, dotenv
