# Crypto Price Analytics Engine
##Overview

A Python-based data pipeline that fetches live cryptocurrency prices from the CoinGecko API, stores them in a MySQL database, and performs 24-hour price analytics.
This project demonstrates real-world API integration, structured data storage, and analytical querying using SQL and Python.

# Tech Stack
- Python
- Requests
- MySQL
- Pandas
- CoinGecko Public API

# Features
 ## Fetch live cryptocurrency prices (Bitcoin, Ethereum)
- Store historical data in MySQL
- Automated data ingestion (runs every 60 seconds)
- 24-Hour Analytics:
- Latest price
- Average price
- Highest price
- Lowest price
